Scripts

Python scripts written mostly by Claude* to process CHEVIE data:

join_lines.py joins broken lines in CHEVIE output
dedup_vectors.py removes duplicate vectors of transformed Hecke character values
permute_vectors.py permutes ordering of transformed character values
mv_multiply.py multiplies vectors by inverses to matrices of generalized Kostka numbers
find_negatives.py identifies negative Haiman-type multiplicities
