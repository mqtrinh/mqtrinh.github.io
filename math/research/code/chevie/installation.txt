GAP3 + CHEVIE installation instructions:

1. Download here: https://webusers.imj-prg.fr/~jean.michel/gap3/
2. Unzip into a directory ./gap (for example).
3. Open gap3-jm/bin/gap.sh, then edit GAP_DIR to be './gap/gap3-jm' (the '.' is CRUCIAL), then save, then close.
4. Ctrl+H, then open .bashrc, then add 'alias gap3='~/gap/gap3-jm/bin/gap.sh' at the end of the file, then save, then close.
5. Open gap3-jm/lib/fptietze.g, then comment out 'DisplayPresentation(T,true);' in line 3265, then save, then close.
6. Open terminal and run 'gap3'.

What is CHEVIE: https://webusers.imj-prg.fr/~jean.michel/chevie/chevie.html

Successful in Ubuntu 22.04, on Lenovo ThinkPad T14s Gen 4 AMD.
