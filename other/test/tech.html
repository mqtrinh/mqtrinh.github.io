HTML

I write webpages (and more) in Atom. In order to avoid retyping the same headers etc. over and over, I write the contents of the webpages in Markdown, then use Pandoc to convert them to HTML in accordance with a template file. To avoid having to choose which webpage to convert each time, I use batch scripts to run the Pandoc commands for multiple webpages with a single click.

Pandoc relies on Python (see below).

LaTeX

I use the MikTeX distribution and the TeXmaker editor. To use Times fonts in both text and math mode, I prefer the font package MTPro2 Lite, available here.

web How to install MathTime Pro fonts on MikTeX.

To actually use MTPro2 Lite in a document, refer to page 4 of the usage manual. It’s the file called mtpro2.pdf in:

mtpro2\doc\fonts\mtpro2

Above, mtpro2 is the name of the directory where you unzipped MTPro2. The key takeaway is: Your accompanying sans-serif font should be Helvetica, rescaled to 92%.

LilyPond

I use the Frescobaldi editor. For text, I use Libre Baskerville and Libre Franklin—the same fonts as on this webpage. For music notation, I use the MTF Arnold font from Music Type Foundry, available for purchase here.

To install new fonts in LilyPond, navigate to:

C:\Program Files (x86)\LilyPond\usr\share\lilypond\current\fonts

Install .ttf and .otf files in the otf directory, and .svg and .woff files in the svg directory. If you install a Music Type Foundry font, then you can find more detailed instructions in a .txt file that they include with the font download.

Windows

If you buy a PC with a fresh copy of Windows 10 installed, and you allow it to connect to the Internet during the initial setup, then you will be forced to create a Microsoft account.
  • To avoid this, skip the step where you are prompted to connect to the Internet.
  • If it already happened and you want to purge the Microsoft account from your computer completely, do a complete reset of the OS.

Use sfc/scannow to find corrupted system files. Starting in Windows 10, you may get the result: “Windows Resource Protection found corrupt files but was unable to fix some of them.” In my case, files called sbcmodel.json and sbcmodel.txt had mismatched hashes. They appear to be related with Windows Update. (To attempt) to fix files like these, run:

Dism /Online /Cleanup-image /Restorehealth

from the command line (in admin mode).

The search bar in Windows 10 is utterly awful. Even the search bar internal to File Explorer is much worse than in Windows 7. To circumvent this, I downloaded the Everything app, available here. In the Search menu of the app, make sure that “Enable Regex” is disabled!

To speed up context menus, etc., in Windows: Open the Registry Editor and navigate to:

HKEY_CURRENT_USER\Control Panel\Desktop

In this directory, set the MenuShowDelay variable to 0.
